# 中国养老方案计算器

一个完全在浏览器本地运行的中文互动计算器，用于比较两种养老情景：

- 参加北京企业职工基本养老保险；
- 将相当于个人养老保险缴费部分的资金投入个人投资账户。

页面可调整工作与退休年龄、缴费基数、增长率、通胀率、投资回报率、养老金调整方式和退休提款方案，并展示工作期积累、退休现金流及个人投资账户耗尽风险。

## 在线使用

GitHub Pages 发布后可访问：

<https://mrpans.github.io/china-pension-calculator/>

也可以直接下载并用浏览器打开 `index.html`。页面不需要服务器、数据库、构建工具或外部 JavaScript 库，也不会上传或保存用户输入。

## 本地预览

```bash
python3 -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 重要说明

本项目只用于养老机制和长期情景比较，不构成投资、税务或法律建议。工资、通胀、投资回报、汇率和政策都可能显著偏离输入假设。模型使用固定长期增长率，不模拟市场波动、收益顺序风险、税费、基金费率或完整的未来政策变化。

模型口径、已知限制和后续工作详见 [HANDOFF.md](HANDOFF.md)。

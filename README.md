# 中国养老方案计算器

一个完全在浏览器本地运行的中文互动计算器，用于比较两种养老情景：

- 参加北京企业职工基本养老保险；
- 将相当于个人养老保险缴费部分的资金投入个人投资账户。

页面可调整工作与退休年龄、缴费基数、增长率、通胀率、投资回报率、养老金调整方式和退休提款方案，并展示工作期积累、退休现金流及个人投资账户耗尽风险。

## 这个模型告诉我们什么

先说结论：这不是一个“社保一定更好”或“自己投资一定更好”的问题。结果主要由两个维度共同决定：**寿命有多长**，以及**个人投资能够长期获得多高的净回报**。

对于多数有正规工作的普通人，持续参保具有很强的基础保障价值。企业职工个人按缴费基数的 8% 缴费并进入个人账户，用人单位另按 16% 缴费进入统筹体系；退休后的养老金通常持续发放，并可能随政策逐年调整。活得越久，统筹养老金承担长寿风险的价值越明显。北京现行缴费比例可参见[海淀区人力社保局说明](https://zyk.bjhd.gov.cn/zwdt/tdzt/2018/csrlsb/cjwtcs/shbxdj/202510/t20251022_4790566_m.shtml)。

相反，如果在退休前去世，或退休后领取时间很短，个人投资通常更有利于留下可继承资产。社保个人账户余额依法可以继承，但单位缴纳的统筹部分不是个人投资账户，不能按个人遗产余额理解；个人账户继承规则可参见[国家发展改革委政策问答](https://www.ndrc.gov.cn/fggz/gbzj/xxyd/202202/t20220225_1317041.html)。

### 两个维度的临界点

下面采用“**家庭累计价值**”口径：

- 社保侧 = 去世前累计领取的养老金 + 去世时尚可继承的个人账户余额；
- 个人投资侧 = 退休后累计提款 + 去世时个人投资账户剩余余额；
- 单位缴纳的统筹基金不作为个人遗产；
- 不计算税务、丧葬补助、抚恤金、基金费率和其他社保待遇。

除寿命和投资回报外，其余参数使用页面默认值：25 岁工作、60 岁退休、当前缴费基数 35,811 元、工资增长 2%、社保个人账户记账利率 1%、养老金每年增长 2%、个人投资提款为社保月收入的 1.3 倍，并假设退休后继续投资。金额均按名义金额逐月计算。

| 去世年龄 | 家庭累计价值相等时的个人投资年化回报率 | 低于临界回报率 | 高于临界回报率 |
|---:|---:|---|---|
| 60 岁 | 约 1.00% | 社保侧较高 | 个人投资侧较高 |
| 65 岁 | 约 2.96% | 社保侧较高 | 个人投资侧较高 |
| 70 岁 | 约 4.33% | 社保侧较高 | 个人投资侧较高 |
| 75 岁 | 约 5.61% | 社保侧较高 | 个人投资侧较高 |
| 80 岁 | 约 6.47% | 社保侧较高 | 个人投资侧较高 |
| 85 岁 | 约 7.01% | 社保侧较高 | 个人投资侧较高 |
| 90 岁 | 约 7.36% | 社保侧较高 | 个人投资侧较高 |
| 95 岁 | 约 7.61% | 社保侧较高 | 个人投资侧较高 |

表格的读法是：如果预计活到 85 岁，且能长期稳定取得高于约 7.01% 的年化回报，那么当前模型中的个人投资家庭累计价值更高；如果长期回报低于约 7.01%，社保侧更高。寿命越长，个人投资需要达到的临界回报率越高，因为社保养老金会继续支付，而个人投资需要持续承担提款。

退休前去世时，比较主要退化为“1% 社保个人账户记账利率”和个人投资回报率的差异。因此，在其他假设不变且个人投资回报高于约 1% 时，个人投资通常能留下更多余额；但这没有计入现实中的遗属待遇等其他保障。

### 账户什么时候会耗尽

在相同默认提款规则下，个人投资账户的耗尽年龄对回报率非常敏感：

| 固定个人投资年化回报 | 模型中的账户耗尽时间 |
|---:|---:|
| 0% | 约 63 岁 6 个月 |
| 2% | 约 65 岁 |
| 4% | 约 67 岁 9 个月 |
| 6% | 约 73 岁 7 个月 |
| 7% | 约 80 岁 3 个月 |
| 7.5% | 约 86 岁 9 个月 |
| 8% | 到 95 岁仍有余额 |

默认固定 8% 回报时，模型计算到 95 岁仍未出现社保侧反超：95 岁时社保侧家庭累计价值约 1,889 万元，个人投资侧约 3,056 万元。但这里的 8% 是每年稳定复利、没有熊市、没有费用和税费的固定收益情景，不能理解成现实中“永远不会枯竭”。真实投资存在波动和收益顺序风险，退休初期的市场下跌尤其可能显著提前耗尽时间。

### 应该怎样理解这个结论

- **寿命较短、重视遗产：**个人投资通常更灵活，且全部剩余资产可以继承；社保侧主要只能继承尚未支付完的个人账户余额，另有遗属待遇但本模型未计入。
- **寿命较长、投资回报一般：**社保的终身现金流更重要。长期回报低于表格对应临界点时，当前模型中社保侧家庭累计价值更高，而且没有个人投资账户先于本人耗尽的问题。
- **投资能力很强：**如果能够在几十年里取得高于对应临界点的净回报，并承受波动、费用、税务和长期纪律要求，个人投资可能到 95 岁仍有余额，并具有更高的家庭累计价值。
- **有正规单位持续缴费：**单位额外缴纳的 16% 和统筹养老金是社保的重要价值来源。对大多数普通人，持续正规就业、按规定参保、满足退休领取条件并为长寿风险做准备，通常是更稳健的基础方案；这不排斥同时进行个人储蓄和投资。

税务会进一步改变临界点。本计算器没有计算个人所得税：社保缴费、工资水平、缴费基数上下限以及用税后收入投资的差异，都可能影响最终结果。因此，这些临界点只适用于当前模型，并不能直接替代个人的养老、投资或税务决策。

## 在线使用

GitHub Pages 发布后可访问：

<https://mrpans.github.io/china-pension-calculator/>

也可以直接下载并用浏览器打开 `index.html`。页面不需要服务器、数据库、构建工具或外部 JavaScript 库，也不会上传或保存用户输入。

## 本地预览

```bash
python3 -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 重要说明

本项目只用于养老机制和长期情景比较，不构成投资、税务或法律建议。工资、通胀、投资回报、汇率和政策都可能显著偏离输入假设。模型使用固定长期增长率，不模拟市场波动、收益顺序风险、税费、基金费率或完整的未来政策变化。

模型口径、已知限制和后续工作详见 [HANDOFF.md](HANDOFF.md)。

# README 寿命与投资回报临界点分析设计

## 目标

在 `README.md` 顶部增加一段基于当前计算器模型的结论，回答寿命长度与个人投资能力如何共同影响“社保养老”和“个人投资养老”的比较结果。

## 比较口径

采用“家庭累计价值”作为二维临界点的主判据：

- 退休前去世：社保侧计个人账户可继承余额，投资侧计个人投资账户全部余额；
- 退休后去世：社保侧计截至该年龄累计领取的养老金，加尚未支付完的个人账户余额；投资侧计累计提款，加去世时个人投资账户剩余余额；
- 单位缴纳的统筹基金不作为个人遗产；
- 不计税费、基金费率、汇率、丧葬补助、抚恤金或其他社保待遇。

现金流安全作为第二判据：个人投资账户一旦耗尽就停止提款，社保养老金按模型设定继续发放至死亡。

## 参数与扫描

除个人投资年化回报率和去世年龄外，使用页面默认参数：25 岁开始工作、60 岁退休、当前缴费基数 35,811 元、工资增长 2%、长期通胀 2.37%、个人账户记账利率 1%、养老金简单年增 2%、计发月数 139、投资提款为社保月收入 1.3 倍、退休后继续投资、使用名义金额。

扫描 60–95 岁的去世年龄与个人投资回报率，求家庭累计价值相等时的临界回报率，并报告若干代表年龄。另报告默认 8% 回报下的寿命临界点和投资账户耗尽状态。

## README 表达

- 先给结论，再给二维临界表；
- 表格中的“低于/高于临界回报率”只表示当前固定收益模型的数值结果；
- 明确“95 岁仍有余额”不等于现实中永不枯竭；
- 强调正规就业单位 16% 缴费、个人 8% 缴费、统筹养老金和个人账户继承的制度差异；
- 将税务、收益波动、收益顺序风险和个人寿命作为决策必须另行考虑的因素；
- 只修改 `README.md` 的产品内容，设计与计划文档除外，不修改页面或计算模型。

# 养老生命周期图合并设计

## 目标

把“工作期资产积累”和“退休后月收入能维持多久”合并成一个完整生命周期图，减少重复卡片，同时避免把账户余额与月收入混在同一纵轴。

## 结构

- 使用一张 `lifecycleChart`，共享从开始工作年龄到95岁的横轴。
- 上半层只覆盖工作期，纵轴为账户余额，展示社保个人账户与标普账户。
- 下半层从退休年龄开始，纵轴为今天购买力（元/月），展示社保持续收入与标普账户耗尽。
- 退休年龄位置贯穿上下两层，使用虚线和“退休”标签标记。
- 删除原有独立的 `growthChart`、`incomeChart` 及其单独卡片。
- 保留回报率敏感性图，因此页面最终包含两张动态图表。

## 验证

- 自动检查只存在 `lifecycleChart` 和 `sensitivityChart` 两张主图。
- 浏览器调整标普回报率后，生命周期图上下层均保持可读，资产路径随参数变化。
- 手机宽度下图表不产生横向溢出，控制台无错误。
